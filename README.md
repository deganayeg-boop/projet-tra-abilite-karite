## Résumé
Ce projet vise à développer un système numérique de suivi des lots de beurre de karité.  
Il permettra de garantir la qualité, la transparence et la conformité aux normes internationales.  
Le dépôt GitHub centralise les documents, rapports et codes sources du projet.
